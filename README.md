# Índice Electrónico del Expediente Judicial

Información General
Herramienta de organización de expedientes judiciales desarrollada para la Rama Judicial - Dirección Seccional de Caldas. Creada por la Oficina de Ejecución y el Centro de Servicios Judiciales de Manizales.

# Funcionalidades

* Ordena expedientes judiciales completos
* Extrae información automática de archivos PDF
* Organiza documentos por fecha y folio
* Identifica archivos protegidos con contraseña
* Clasifica documentos como Digitalizados o Electrónicos

# Macro de Complementación (Macro3)

* Agrega nuevos documentos a índices existentes
* Mantiene la información ya registrada
* Procesa solo archivos nuevos
* Compatible con trabajos anteriores
